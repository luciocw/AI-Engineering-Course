# Guia de Setup

## Ferramentas Necessarias

### Node.js 20+
```bash
# Verifique versao
node -v

# Instale via nvm (recomendado)
nvm install 20
nvm use 20
```

### Git
```bash
git config --global user.name "Seu Nome"
git config --global user.email "seu@email.com"
```

### API Keys

1. **Anthropic:** Crie conta em console.anthropic.com ($5 gratis)
2. **LangSmith:** Crie conta em smith.langchain.com (gratis)
3. **Qdrant Cloud:** (opcional) cloud.qdrant.io (gratis)
