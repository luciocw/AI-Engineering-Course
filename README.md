> Curso pratico de AI Engineering: construa 4 repositorios de portfolio enquanto aprende as skills mais pedidas no mercado.

## O que voce vai construir

| Dia | Projeto | Skills |
|-----|---------|--------|
| 1 | `ai-prompt-engineering-toolkit` | Handlebars, Claude API, Tool Use, Data Pipelines |
| 2 | `rag-knowledge-assistant` | Embeddings, Vector DB, RAG Pipeline |
| 3 | `multi-agent-ai-system` | Multi-agent, Tools, Streaming |
| 4 | `production-ai-service` | Docker, CI/CD, LangSmith, Deploy |

## Pre-requisitos

- Node.js 20+
- Git configurado
- Conta Anthropic (API key) - $5 gratis
- Conta LangSmith (gratis ate 5k traces/mes)
- Docker Desktop (Dia 2+)

## Como Usar

1. Leia o README.md de cada modulo
2. Execute os exercicios na ordem (`ex1`, `ex2`, ...)
3. Tente resolver antes de ver as solucoes em `/solucoes`
4. Complete o desafio final de cada modulo
5. Commit com mensagem convencional e push para seu GitHub

## Stack

- **Runtime:** Node.js 20 + TypeScript
- **LLM:** Claude API (Sonnet 4.5, Haiku 4.5)
- **Templates:** Handlebars
- **Vector DB:** Qdrant / Pinecone
- **Observability:** LangSmith
- **Deploy:** Docker + GitHub Actions + Railway
