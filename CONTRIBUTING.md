# Contribuindo

Obrigado pelo interesse em contribuir com o curso!

## Como Contribuir

1. Fork o repositorio
2. Crie uma branch: `git checkout -b feat/minha-contribuicao`
3. Faca suas alteracoes
4. Commit com mensagem convencional: `git commit -m "feat(modulo): descricao"`
5. Push: `git push origin feat/minha-contribuicao`
6. Abra um Pull Request

## Commits Convencionais

```
feat:     Nova funcionalidade
fix:      Correcao de bug
docs:     Documentacao
test:     Testes
refactor: Refatoracao
chore:    Tarefas gerais
```

## Reportando Bugs

Abra uma Issue com:
- Descricao do problema
- Passos para reproduzir
- Comportamento esperado vs atual
- Versao do Node.js e SO

## Sugerindo Melhorias

Use GitHub Discussions para sugestoes de novos exercicios, modulos ou melhorias.
